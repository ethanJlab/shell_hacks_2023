### Running the database
Navigate to the directory 
```
../shellhacks2023/pocketbase_0.18.6
```
run the command
```
./pocketbase serve
```
The app should run on ```localhost:8090```

### Running the backend server
Navigate to the directory
```
../shellhacks2023/backend
```
run the command
```
npm i
npm start
```
The app should run on ```localhost:5000```
